# Gitee

Git 全局设置:

git config --global user.name "天使之翼"
git config --global user.email "553777402@qq.com"

创建 git 仓库:

mkdir note
cd note
git init 
touch README.md
git add README.md
git commit -m "first commit"
git remote add origin https://gitee.com/zuqiuxiaojiang/note.git
git push -u origin "master"

已有仓库?

cd existing_git_repo
git remote add origin https://gitee.com/zuqiuxiaojiang/note.git
git push -u origin "master"



# 1. 更新软件源
pkg update && pkg upgrade

# 2. 安装 Git
pkg install git

# 3. 授予 Termux 访问手机存储的权限
termux-setup-storage
###### 执行后会弹出权限请求，点击允许

# 4. 配置 Git 全局信息（与 Gitee 账号一致）
git config --global user.name "你的Gitee用户名"
git config --global user.email "你的Gitee注册邮箱"


# 进入内部存储
cd storage/shared

# 创建 Obsidian 目录（可选）
mkdir Obsidian && cd Obsidian

# 克隆 Gitee 仓库（使用 HTTPS + 私人令牌格式）
git clone https://<用户名>:<私人令牌>@gitee.com/<用户名>/<仓库名>.git

示例：
git clone https://zhangsan:5b026888e5a7a35edefbd77033d7808f@gitee.com/zhangsan/my-notes.git

git@gitee.com:zuqiuxiaojiang/note.git

git clone https://zuqiuxiaojiang:b289b793c879211cde476eefcf668900@gitee.com/zuqiuxiaojiang/note.git
